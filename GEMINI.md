# Garden Madness

This repository contains a digital garden built with Astro.

## Current State

*   **Framework:** Astro
*   **Styling:** Tailwind CSS
*   **Linting:** ESLint
*   **Formatting:** Prettier

## Dependencies

*   `astro`
*   `tailwindcss`
*   `@tailwindcss/vite`

*   `eslint-plugin-jsx-a11y`
*   `eslint-plugin-prettier`
